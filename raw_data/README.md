# 原始数据文件夹

该文件夹用于存储项目的原始数据集文件。

## 预期内容

这个文件夹应包含：

- 足球比赛视频序列的原始图像帧
- 相关的标注信息文件（如txt、json格式）
- 数据集描述文档
- 原始文件目录结构

## 数据格式

原始数据应按以下结构组织：

```
raw_data/
├── video1/
│   ├── img1/
│   │   ├── 000001.jpg
│   │   ├── 000002.jpg
│   │   └── ...
│   └── gt/
│       └── gt.txt
├── video2/
│   ├── img1/
│   └── gt/
└── ...
```

其中，`gt.txt`文件包含每帧图像中目标的边界框、ID和类别信息。

## 注意事项

- 原始数据不应直接修改
- 数据处理应通过`data_processor.py`脚本进行
- 处理后的数据将保存在`processed_data`文件夹中
